Implementează un algoritm care citește de la tastatură un număr natural n și un mesaj, care poate conține spații.\
Programul va afișa mesajul de n ori, fiecare afișare pe câte o linie nouă și însoțită de numărul curent, ca în exemplul următor:

Console

    # INPUT:
    4
    Programming is really fun!
    # OUTPUT:
    1. Programming is really fun!
    2. Programming is really fun!
    3. Programming is really fun!
    4. Programming is really fun!
